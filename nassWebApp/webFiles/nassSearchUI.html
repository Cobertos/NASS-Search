NHTSA NASS Search Tool

This is an advanced tool to search the National Highway Transportation and Safety Administration(NHTSA) National Automotive Sampling System(NASS) database. This tool provides searching on any database column and special data export not found in the original NASS case viewer / search engine.

This site is in no way affiliated with the US government or NHTSA.

To perform a search, simply:

  • Build a search by clicking the panel below
  • Select your year range and export format (if a file is desired for export)
  • Click the 'Go' button to begin
